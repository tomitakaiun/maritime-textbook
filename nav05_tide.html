5-1. 潮汐とは何か

潮汐（ちょうせき）とは、主に月と太陽の引力（起潮力）によって海面が周期的に上下する現象です。

起潮力のしくみ

月の引力は地球の月に面した側で最も強く、反対側で最も弱くなります。この引力の差（起潮力）により、地球を取り巻く海水は月に面した側と反対側の2か所で膨らみ、楕円形に変形します。

  • 月に面した側 — 月の引力が遠心力を上回り、海水が月側に引っ張られる
  • 月の反対側 — 遠心力が月の引力を上回り、海水が外側に押し出される
  • その結果、地球は約24時間50分で自転する間に、ほぼ2回の高潮（こうちょう）と2回の低潮（ていちょう）を経験します

太陽も起潮力を持ちますが、距離が遠いため月の約46%程度です。月と太陽の位置関係によって潮汐の大きさが変化します。

5-2. 大潮と小潮

潮汐の大きさ（潮差）は、月と太陽の位置関係によって周期的に変化します。

大潮（Spring Tide）

大潮は、月と太陽が一直線上に並ぶとき（朔=新月、望=満月）に発生します。月と太陽の起潮力が合算されるため、潮差が最大になります。

小潮（Neap Tide）

小潮は、月と太陽が直角の位置関係にあるとき（上弦・下弦の月）に発生します。月と太陽の起潮力が相殺し合い、潮差が最小になります。

図5-2: 月の位相と大潮・小潮の関係

大潮の時期は潮差が大きく、潮流も強くなります。内航タンカーの航海計画では、大潮期の狭水道通過や浅海域での入港に特に注意が必要です。逆に小潮の時期は潮差・潮流ともに穏やかで、操船がしやすくなる傾向があります。

潮汐の周期

周期説明所要時間
半日潮周期高潮から次の高潮（主要な周期）約12時間25分
1日潮周期潮汐の日周不等が卓越する場合約24時間50分
大潮-小潮周期大潮から次の大潮約14.8日（半月）

日本沿岸では半日潮（1日に2回の高潮・低潮）が卓越しますが、地域によっては1日1回しか顕著な高潮が生じない日潮不等が大きい場所もあります。

基本水準面（CDL）と海図水深の関係

海図に記載されている水深は、基本水準面（CDL）から測った深さです。CDLは「ほぼ最低水面」に設定されているため、実際の水深は海図水深より大きい（深い）ことがほとんどです。潮汐表で求めた潮高を海図水深に加えると、その時刻の実際の水深（概算）が得られます。

実際の水深 = 海図水深 + 潮高

5-4. 潮汐表の読み方

標準港と副港

潮汐表は、主要な港（標準港）の潮時と潮高を本表に掲載しています。それ以外の港（副港）は、潮時差と潮高比を使って標準港のデータから計算で求めます。

潮時差と潮高比

  • 潮時差 — 標準港の高潮（低潮）時刻に加減して副港の潮時を求める補正値
  • 潮高比 — 標準港の潮高に乗じて副港の潮高を求める係数

計算例: 副港の潮時・潮高を求める

条件:

  • 標準港: 広島
  • 副港: 某港（潮時差: HW +0時間25分、LW +0時間15分、潮高比: HW 0.85、LW 0.90）
  • 広島の潮汐予報: HW 06:30（潮高 320cm）、LW 12:45（潮高 50cm）

計算:

  • 副港のHW時刻 = 06:30 + 0:25 = 06:55
  • 副港のHW潮高 = 320 × 0.85 = 272cm
  • 副港のLW時刻 = 12:45 + 0:15 = 13:00
  • 副港のLW潮高 = 50 × 0.90 = 45cm

内航タンカーの入港計画では、着桟予定時刻の潮高を必ず確認します。特に浅い港では、低潮時に喫水が不足する可能性があるため、「海図水深 + 潮高 - 喫水 ≥ 安全余裕水深（UKC）」を確認することが不可欠です。

潮汐曲線の利用

高潮と低潮の間の時刻の潮高を求めるには、潮汐曲線（タイダルカーブ）を利用します。潮汐表に付属する標準的な潮汐曲線を使い、高潮・低潮の時刻と潮高から内挿して任意の時刻の潮高を推定します。

5-5. 潮流とは何か

潮流とは、潮汐に伴って生じる海水の水平方向の周期的な流れです。潮汐が海面の上下運動であるのに対し、潮流は水の移動そのものです。

海峡や水道など狭い場所では、潮汐による海面差が大きくなり、その差を解消しようとして強い潮流が発生します。内航航海では潮流の把握が極めて重要です。

5-6. 潮流の種類

往復流（Reversing Current）

海峡や水道で見られる、一定の方向に流れた後、方向が反転する潮流です。日本の多くの海峡では往復流が卓越します。

  • 東流（上げ潮流）と西流（下げ潮流）のように、2方向を往復する
  • 流向の転換時に流速が一時的にゼロになる（転流）
  • 来島海峡、明石海峡、関門海峡などが代表例

回転流（Rotary Current）

開けた海域で見られる、流向が時計回り（または反時計回り）に連続的に変化する潮流です。流速はゼロにならず、常に一定方向に回転しています。

5-7. 潮流表と潮流図の使い方

潮流表

潮流表は、主要な海峡や水道の潮流の転流時刻と最強流速を予報した表です。使い方は潮汐表と似ています。

  • 標準地点の転流時刻と最強流速が本表に掲載される
  • その他の地点は流時差と流速比で補正して求める
  • 転流の時刻は必ずしも高潮・低潮の時刻とは一致しない

潮流図（タイダルストリーム・アトラス）

潮流図は、ある海域の潮流の方向と速さを、高潮または低潮の前後何時間かで表した図です。矢印の方向が流向、矢印の太さや数字が流速を示します。

注意: 潮流表・潮流図の限界

潮流表や潮流図はあくまで予報値（平均的な値）です。風や気圧の変化で実際の潮流は予報値と大きく異なることがあります。特に台風接近時や強い季節風の時期は注意が必要です。

5-8. 内航航海と潮汐・潮流

入港時の水深確認

内航タンカーは積荷の量によって喫水が大きく変化します。満載時の入港では以下の確認が必須です。

実務例: 浅い港への入港判断

条件:

  • 本船喫水: 5.5m（満載）
  • 入港先バース前面の海図水深: 6.0m（CDL基準）
  • 入港予定時刻の潮高: 1.2m
  • 会社規定の安全余裕水深（UKC）: 1.0m

計算:

  • 実際の水深 = 海図水深 + 潮高 = 6.0 + 1.2 = 7.2m
  • UKC = 実際の水深 - 喫水 = 7.2 - 5.5 = 1.7m
  • 1.7m ≥ 1.0m（規定UKC）→ 入港可能

ただし、低潮時（潮高0.3mの場合）は水深6.3m、UKC=0.8mとなり規定を満たしません。入港時刻の選定が重要です。

狭水道通過時の潮流計算

瀬戸内海の狭水道を通過する際は、潮流を利用した効率的な航海計画が重要です。

実務例: 来島海峡の通過計画

来島海峡は順中逆西の航法が適用されます。潮流の方向によって使用する水道が変わるため、通過時の潮流を正確に把握する必要があります。

  • 南流（S流）時: 中水道を北から南へ通過（順潮）
  • 北流（N流）時: 西水道を南から北へ通過（順潮）
  • 転流前後の弱流時を狙うか、順潮に乗るのが基本
  • 最強潮流は10ノットを超えることもあり、逆潮での通過は非常に困難

内航タンカーの航海士は、航海計画を立てる際に潮汐表・潮流表を必ず確認し、入港時の水深、狭水道の通過時刻を潮汐・潮流と照らし合わせて決定します。これは安全航海の基本中の基本です。

5-9. 瀬戸内海の潮流特性

瀬戸内海は多数の島嶼と狭い海峡からなり、潮流が複雑で強い海域です。内航タンカーが特に注意すべき海域を解説します。

来島海峡

  • 最強流速: 約10ノット（大潮時）
  • 日本三大急潮流のひとつ
  • 順中逆西の特殊航法 — 海上交通安全法で規定
  • 3つの水道（西水道・中水道・東水道）があり、潮流方向により使い分け
  • 潮流信号所（中渡島・馬島）の情報を必ず確認

鳴門海峡

  • 最強流速: 約8〜10ノット
  • 世界的にも有名な渦潮が発生
  • 可航幅が狭く、大型船は通過を避けることが多い
  • 内航タンカーでも小型船以外は迂回ルートを検討

明石海峡

  • 最強流速: 約6〜7ノット
  • 交通量が非常に多く、航路が設定されている
  • 潮流に加え、大型船が多いため見張りの強化が必須

関門海峡

  • 最強流速: 約4〜5ノット（早鞆瀬戸付近）
  • S字状に曲がった航路で、見通しが悪い
  • 関門海峡海上交通センター（関門マーチス）による管制あり
図5-5: 瀬戸内海主要海峡の最強潮流比較（大潮時の概算値）

実務上の注意: 潮流と操船

内航タンカーの多くは航海速力が10〜12ノット程度です。来島海峡の最強潮流（約10ノット）に逆らって航行すると、対地速力がほぼゼロになるか、押し戻されることがあります。必ず順潮時に通過するか、転流時を狙って計画を立てましょう。逆潮での無理な通過は衝突・乗揚げ事故の原因となります。

内航タンカー実務: 潮汐・潮流の情報源

  • 潮汐表・潮流表（海上保安庁刊行） — 年1回発行。最も基本的な資料
  • 潮汐推算（海上保安庁Webサイト） — インターネットで任意の日時の潮汐を閲覧可能
  • 来島海峡潮流情報（海上保安庁） — 来島海峡の潮流予報をリアルタイム配信
  • MIRC（航海用電子参考図） — ECDISやPCで潮流情報を表示
  • AIS航行支援情報 — 潮流信号所のデータがAIS経由で受信可能

章末確認クイズ

Q1. 大潮はいつ発生しますか？ その理由も含めて説明してください。

新月（朔）と満月（望）の頃に発生します。この時、月と太陽が地球に対して一直線上に並ぶため、両者の起潮力が合算されて潮差が最大になります。約14.8日（半月）ごとに大潮が繰り返されます。

Q2. 海図水深5.0m、潮高1.5mの場所に喫水5.8mの船が入港できますか？ UKCを0.5m以上確保する必要がある場合で判断してください。

入港可能です。実際の水深 = 5.0 + 1.5 = 6.5m。UKC = 6.5 - 5.8 = 0.7m。規定の0.5m以上を確保できています。ただし、潮高は時間とともに変化するため、在泊中に低潮を迎えた場合の水深も確認が必要です。

Q3. 来島海峡で適用される「順中逆西」とはどういう意味ですか？

来島海峡には3つの水道（西水道・中水道・東水道）がありますが、潮流の方向によって使用する水道が決められています。「順中逆西」とは、順潮の場合は中水道を、逆潮の場合は西水道を通航するという原則です。具体的には、南流時は中水道を北→南へ（順潮）、北流時は西水道を南→北へ（順潮）通過します。これにより対向船との衝突リスクを減らしています。
