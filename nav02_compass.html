第2章 コンパスと方位

内航船員のための海技教材 — 航海編
目次へ « 前章: 船の基礎知識 次章: 海図の読み方 »

船が安全に目的地へ到達するには、自分がどの方向に進んでいるかを常に知っておく必要があります。陸上では道路標識や建物を目印にできますが、海上では周囲が水平線だけということも珍しくありません。そこで不可欠なのがコンパス（羅針儀）です。本章では、方位の基礎概念と各種コンパスの仕組み、そして方位の換算方法を学びます。

2-1. 方位の基礎

方位とは「ある方向が北からどれだけの角度にあるか」を示す数値です。0度（北）から時計回りに360度で一周します。しかし、船の世界では「北」が3種類あるため、注意が必要です。

3つの「北」

名称英語記号説明
真北True NorthT地球の北極点（地理上の北極）の方向。海図の基準
磁北Magnetic NorthM磁気コンパスの針が指す方向。地磁気の北極の方向
コンパス北Compass NorthC船上のコンパスが実際に指す北。船体磁気の影響あり

これら3つの北は、通常一致しません。その差を偏差（Variation）と自差（Deviation）と呼びます。

方位の種類を覚える語呂合わせ: 「真（しん）→ 偏（へん）→ 磁（じ）→ 自（じ）→ コ（こ）」
真方位 →（偏差を加減）→ 磁針方位 →（自差を加減）→ コンパス方位
「TVは（V=Variation）Mだけ（M=Magnetic）、Dは（D=Deviation）Cだけ（C=Compass）」と覚える人もいます。

図2-1: 方位盤。北(000°)から時計回りに360度。オレンジの線は045度（北東）方向の例

32方位と度数表示

古くは方位を32に分割して呼んでいましたが（N, NNE, NE, ENE, E...）、現代の航海では3桁の度数表示（000°〜359°）を使います。「北」は000度、「東」は090度、「南」は180度、「西」は270度です。方位は必ず3桁で表記します（例: 5度は「005」）。

2-2. 磁気コンパスの構造と原理

磁気コンパスは、電源を必要とせず、地球の磁力線を利用して方位を示す最も基本的な航海計器です。電子機器が故障しても使えるため、すべての船に装備が義務付けられています。

構造

  • コンパスボウル: コンパスを収納する球形の容器。液体で満たされている
  • 磁針（コンパスカード）: 磁石が取り付けられた目盛り盤。液体中で自由に回転する
  • ピボット: カードの中心を支える軸受け。摩擦を最小限にする
  • 液体: アルコールと水の混合液。カードの動きを安定させ（制振）、ピボットの荷重を軽減する
  • ラバーズライン: ボウルに刻まれた基準線。船首方向を示す
  • ジンバル: 船の動揺があってもコンパスを水平に保つ装置
  • ビナクル（羅針台）: コンパスを載せる台。自差修正用の磁石や軟鉄球を内蔵
図2-2: 磁気コンパスの断面構造。液体中で磁針（コンパスカード）が自由に回転し、磁北を指す

なぜ液体が入っているのか

コンパスボウル内の液体には2つの重要な役割があります。(1) 制振作用: 船の動揺でカードが暴れるのを抑え、安定した指示を得る。(2) 浮力: カードを浮かせてピボットにかかる重さを減らし、摩擦による誤差を最小にする。寒冷地では凍結防止のためアルコール濃度が調整されています。

2-3. 偏差と自差

磁気コンパスが示す方位は「真の方位」とは一致しません。その差を生む2つの要因を理解しましょう。

偏差（Variation）

偏差とは、真北と磁北のずれの角度です。地球の磁極は地理上の北極と一致しておらず、しかも年々移動しています。偏差は場所によって異なり、海図上のコンパスローズに記載されています。

  • 日本近海では一般に西偏 6°〜9°程度（場所による）
  • 偏差は毎年少しずつ変化する（年差と呼ぶ）
  • 東偏（E）= 磁北が真北より東にある場合 → プラス(+)
  • 西偏（W）= 磁北が真北より西にある場合 → マイナス(-)

自差（Deviation）

自差とは、船体が持つ磁気がコンパスに及ぼす影響で生じるずれです。船は鋼鉄でできているため、船体自体が磁石のように振る舞い、コンパスの指示に影響を与えます。

  • 自差は船首方位（ヘディング）によって変化する
  • 各方位ごとの自差は自差表（Deviation Table）にまとめられている
  • 自差修正はビナクル内の修正磁石や軟鉄球で行う（コンパスアジャスト）
  • 新しい鋼材を積んだり、荷役後など、自差は変化することがある

2-4. 方位の換算

航海では、海図上の方位（真方位）とコンパスの読み（コンパス方位）を相互に変換する必要があります。6級海技士試験の定番問題でもあります。

換算の公式

真方位(T) = コンパス方位(C) + 自差(D) + 偏差(V)

逆に: コンパス方位(C) = 真方位(T) - 偏差(V) - 自差(D)

途中: 磁針方位(M) = 真方位(T) - 偏差(V)

※ 東偏(E)/東自差(E) はプラス(+)、西偏(W)/西自差(W) はマイナス(-) として計算

計算例1: 真方位 → コンパス方位

問題

海図上で目的地への真方位が135°(T)である。偏差7°W、この船首方位での自差3°Eのとき、コンパスで何度にすればよいか。

解答

偏差 7°W = -7、自差 3°E = +3

C = T - V - D = 135 - (-7) - (+3) = 135 + 7 - 3 = 139°(C)

コンパスで139度に針路をとればよい。

計算例2: コンパス方位 → 真方位

問題

コンパスで灯台の方位を測定したところ220°(C)であった。偏差8°W、自差2°Wのとき、真方位はいくつか。

解答

偏差 8°W = -8、自差 2°W = -2

T = C + D + V = 220 + (-2) + (-8) = 220 - 2 - 8 = 210°(T)

海図上で210度の方向に灯台がある。

計算の注意点

  • 結果が360以上になったら360を引く（例: 365 → 005）
  • 結果がマイナスになったら360を足す（例: -5 → 355）
  • 東(E)はプラス、西(W)はマイナス。この符号を間違えると逆方向になるので要注意

計算例3: 内航タンカーの実務例

場面: 裕洋丸が千葉港を出港し、東京湾を南下中

海図上の計画針路は200°(T)。千葉沖の偏差は7.5°W（コンパスローズに記載）。この針路での自差は自差表から1°E。

コンパス針路: C = 200 - (-7.5) - (+1) = 200 + 7.5 - 1 = 206.5° ≒ 207°(C)

操舵手に「コース207度」と指示する。

2-5. ジャイロコンパスの概要

ジャイロコンパスは、高速回転するジャイロスコープ（コマ）の性質を利用して真北を指す計器です。磁気コンパスと異なり、偏差や自差の影響を受けません。

磁気コンパスとの比較

項目磁気コンパスジャイロコンパス
指す方向磁北（偏差+自差あり）真北（ジャイロエラーのみ）
電源不要必要
起動時間即時数時間（コマの安定まで）
偏差・自差影響あり（換算が必要）影響なし
故障リスク極めて低い電源喪失時に使用不可
レピータなし各所にレピータ（子機）を設置可
コスト安価高価

ジャイロコンパスの原理（簡潔に）

高速回転するコマには「回転軸の方向を保とうとする性質（剛性）」と「外力を受けると90度ずれた方向に軸が移動する性質（歳差運動）」があります。この性質を地球の自転と組み合わせることで、コマの軸が自動的に真北を指すように設計されています。詳しい物理は上級で学びますが、6級レベルでは「電源が必要で真北を指す」「磁気の影響を受けない」ことを覚えれば十分です。

ジャイロコンパスにもジャイロエラーと呼ばれる誤差があります。速力や緯度の変化によって生じるもので、通常1〜2度以内です。内航船では航海距離が短いため、影響は小さいですが、定期的な確認は必要です。

2-6. GPSコンパスの概要

近年の内航船にはGPSコンパスが搭載されることが増えています。2本以上のGPSアンテナの位置差から船首方位を算出する装置です。

GPSコンパスの特徴

  • GPS衛星からの信号で真方位を表示する
  • 磁気の影響を全く受けない（偏差・自差なし）
  • 起動が早い（GPS信号を受信すれば数分で使用可能）
  • ジャイロコンパスより安価で、小型船にも搭載しやすい
  • GPS信号が遮断される状況（ジャミング等）では使用不可
  • ヘディングセンサーとしてレーダーやAISにも方位情報を提供

コンパスの冗長性（バックアップ）

電子機器は故障する可能性があります。GPSコンパスやジャイロコンパスが使えなくなった場合のバックアップとして、磁気コンパスは必ず装備されています。磁気コンパスの使い方と方位換算の知識は、電子機器全盛の時代でも船員の必須スキルです。内航タンカーでは、狭い航路を航行することが多いため、コンパスの故障は直ちに安全に影響します。

2-7. コンパスエラーの点検方法

コンパスが正しい値を指しているかどうかを定期的に確認することをコンパスチェック（コンパスエラーの点検）と呼びます。内航船でも毎航海、少なくとも1日1回は行うべき重要な作業です。

主な点検方法

1. 陸上物標による方位比較

海図上で位置のわかっている灯台などの物標について、コンパスで測定した方位と海図から求めた真方位を比較します。

手順

  1. 自船の位置を確定する（GPS等）
  2. 海図上で自船位置から物標までの真方位を読み取る → 例: 090°(T)
  3. コンパスで同じ物標の方位を測定する → 例: 096°(C)
  4. 差を計算: コンパスエラー = T - C = 090 - 096 = -6°（6°W）
  5. 偏差(V)を引いて自差(D)を確認: D = エラー - V = -6 - (-7) = +1 = 1°E
  6. 自差表の値と照合し、異常がないか確認する

2. 太陽方位による点検

太陽の方位は天測暦（天測計算表）やGPS計算で正確に求められます。日出・日没時の太陽方位は計算が容易で、コンパスチェックによく利用されます。

3. ジャイロコンパスとの比較

ジャイロコンパスを搭載している船では、ジャイロ方位と磁気コンパス方位を比較することで、簡易的にチェックできます。ただし、ジャイロ自体のエラーもあり得るため、陸上物標での確認も併用すべきです。

4. GPSコンパスとの比較

GPSコンパスは真方位を直接表示するため、磁気コンパスとの差からコンパスエラー（偏差+自差）を直接読み取れます。内航船では最も手軽な方法です。

図2-4: コンパスチェックの手順。海図から求めた真方位とコンパス実測値を比較する

コンパスチェックは航海安全の基本動作です。特に内航タンカーでは狭い港湾・航路を航行することが多く、コンパスの誤差が大きいと航路逸脱や座礁のリスクが高まります。毎航海必ず実施し、異常値が出たら速やかに船長に報告しましょう。自差が急に大きく変わった場合、コンパスアジャスターによる自差修正（コンパスアジャスト）が必要です。

まとめ

  • 方位は北から時計回りに360度で表す。3桁表記（例: 005°）
  • 「北」は3種類ある: 真北(T)、磁北(M)、コンパス北(C)
  • 偏差(V)は場所による真北と磁北の差、自差(D)は船体磁気による磁北とコンパス北の差
  • 方位の換算: T = C + D + V（東=+、西=-）
  • 磁気コンパスは電源不要で信頼性が高い。すべての船に必須
  • ジャイロコンパスは真北を指すが、電源が必要
  • GPSコンパスは安価で起動が早く、内航船に普及している
  • コンパスエラーの点検は毎航海必ず行う

章末クイズ

Q1. 真方位150°、偏差7°W、自差2°Eのとき、コンパス方位はいくつか？

  • A. 141°
  • B. 145°
  • C. 155°
  • D. 159°

正解: C. 155°

C = T - V - D = 150 - (-7) - (+2) = 150 + 7 - 2 = 155°

偏差が西偏(W)なので符号はマイナス、自差が東(E)なので符号はプラスです。

Q2. 磁気コンパスのボウル内に液体が入っている理由として正しいものはどれか？

  • A. 磁針の磁力を強くするため
  • B. カードの動きを安定させ（制振）、ピボットの荷重を軽減するため
  • C. コンパスの温度を一定に保つため
  • D. 防錆のため

正解: B. カードの動きを安定させ（制振）、ピボットの荷重を軽減するため

液体はコンパスカードに浮力を与えてピボットの摩擦を減らし、また船の動揺によるカードの振動を抑制します（制振作用）。

Q3. コンパス方位が260°(C)、偏差8°W、自差3°Wのとき、真方位はいくつか？

  • A. 249°
  • B. 255°
  • C. 265°
  • D. 271°

正解: A. 249°

T = C + D + V = 260 + (-3) + (-8) = 260 - 3 - 8 = 249°

偏差・自差ともに西(W)なのでマイナスとして計算します。
